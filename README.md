## 🎵 O que é o Windsound?

**Windsound** é um player de música em desenvolvimento que utiliza a **API do Innertube** (a API interna do YouTube) para transmitir e reproduzir músicas diretamente do YouTube. Com login via conta do YouTube, o Windsound oferece uma experiência de escuta personalizada e integrada ao maior catálogo de música do mundo.

O projeto nasceu com a ideia de criar uma interface musical limpa e intuitiva, aproveitando o enorme acervo disponível no YouTube, sem depender de serviços de streaming pagos.

---

## ✨ Funcionalidades (Beta)

- 🔐 **Login com YouTube** — Autenticação via conta Google/YouTube
- 🎶 **Reprodução de músicas** — Streaming de áudio via API Innertube
- 🔍 **Busca integrada** — Pesquise músicas, artistas e álbuns diretamente pelo YouTube
- 📋 **Filas de reprodução** — Organize e gerencie o que vai tocar a seguir

> 🚧 Mais funcionalidades estão sendo desenvolvidas
